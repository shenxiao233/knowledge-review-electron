# 知识管理与复习工具 使用说明书

## 1. 软件定位

知识管理与复习工具是一款 Electron 桌面端学习应用原型，首页采用语雀式知识库工作台布局，以紧凑主导航、文档树、正文编辑器、大纲和复习侧栏组织核心流程，并整合选择题制卡、间隔重复复习、卡片库、回收站、个人中心和设置管理。

## 2. 启动方式

在项目目录执行：

```bash
npm install --cache .npm-cache
npm run doctor
npm run dev
```

`npm run doctor` 用于检查 Electron 命令、Electron 运行时、主进程和页面文件是否存在。

## 3. 主界面导航

左侧竖向导航栏包含以下入口：

- 文档库：编辑学习笔记，并在右侧查看大纲和复习侧栏。
- 卡片库：查看卡组、筛选卡片、导出卡片、新建选择题卡片。
- 复习：进入居中专注复习模式。
- 回收站：查看空状态和分类标签页。
- 插件市场：当前版本暂时隐藏，避免未实现功能干扰主流程。
- 个人中心：查看学习数据、卡片数量、累计复习和打卡热图。
- 设置：调整外观入口、复习算法参数和存储相关操作。

## 4. 文档库使用

文档库页面由三部分组成：

- 左侧文档树：展示文件夹和文档层级。
- 中间编辑器：支持标题、段落、加粗、斜体、下划线、删除线、高亮、列表、引用、分割线和链接等基础富文本编辑。
- 右侧区域：包含大纲目录和卡片复习侧栏，两栏均可独立折叠，折叠时正文区域会自动扩展。

使用方式：

1. 在中间编辑器直接输入或修改内容。
2. 使用顶部格式栏调整段落、标题、文字样式和列表。
3. 点击“保存”保存文档到本地存储。
4. 选中文本后，底部浮动工具条会出现，可点击“提取高亮”或“生成问答”。
5. 点击顶部“制卡”可基于已选中文本快速生成卡片。
6. 左侧默认提供《间隔重复学习法》《高效阅读与知识整理》《React Hooks 核心概念》三篇测试文档；已有用户数据不会被覆盖。

## 5. 自定义制卡

在卡片库或复习页点击“新建卡片 / 制卡”进入制卡弹窗。

必填字段：

- 题干
- 选项 A
- 选项 B
- 选项 C
- 选项 D
- 正确答案

选填字段：

- 解析
- 标签

保存后，卡片会写入本地存储，并加入复习队列。

## 6. 卡片库使用

卡片库页面包含：

- 顶部卡组概览：显示不同卡组的卡片数量和待复习数量。
- 卡组交互：点击卡组卡片会自动筛选下方卡片列表；“新建卡组”会创建并保存一个新的卡组名称。
- 搜索框：按卡组名称或卡片题干筛选卡组。
- 卡片列表：支持按文件夹和标签筛选。
- 导出按钮：支持导出选中卡片或当前文件夹。

导出格式支持：

- JSON
- Markdown

## 7. 复习使用

复习分为两种模式：

- 文档库右侧复习侧栏：适合边看笔记边刷卡。
- 复习页专注模式：适合全屏集中答题。

答题逻辑：

1. 系统展示题干和 A/B/C/D 四个选项。
2. 点击选项后立即判定。
3. 答对：正确选项显示绿色。
4. 答错：所选选项显示红色，正确答案显示绿色。
5. 判定后自动展示解析。
6. 点击“下一题”进入后续卡片。

## 8. 间隔重复规则

当前实现了轻量间隔重复逻辑：

- 到期卡片进入复习队列。
- 答错卡片会在当天再次出现。
- 答对卡片会根据掌握度和遗忘曲线系数延长复习间隔。
- 复习队列按到期时间排序。
- 每日复习上限可在设置中调整。

## 9. 打卡热图

打卡热图展示最近 52 周复习记录。

颜色规则：

- 灰色：0 题
- 浅绿色：1-10 题
- 中绿色：11-30 题
- 深绿色：30 题以上

鼠标悬停可查看日期和当天复习数量。

## 10. 导入导出

顶部“导入”按钮可选择本地 JSON 或 Markdown 文件导入卡片。

导入规则：

- 导入卡片新增到卡片库。
- 不覆盖已有卡片。
- 导入时会校验题干、选项和正确答案字段。

导出规则：

- 支持导出选中卡片。
- 支持导出当前文件夹。
- 支持导出全部卡片。
- 支持 JSON 和 Markdown 两种格式。

## 11. 设置

设置页面包含：

- 外观：卡片样式和字体大小入口。
- 复习算法：遗忘曲线系数、每日复习上限和间隔预览。
- 存储：说明本地存储策略，并提供导出全部数据入口。
- 关于：软件说明。
